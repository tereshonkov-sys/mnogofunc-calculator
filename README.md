# 🧮 Universal Pygame Calculator

Многофункциональное десктопное приложение-калькулятор, разработанное на **Python** с использованием библиотеки **Pygame**. Приложение включает в себя не только стандартные арифметические операции, но инструменты для работы с системами счисления, систему регистрации пользователей и персонализацию интерфейса.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Pygame](https://img.shields.io/badge/Pygame-2.0+-red.svg)

## ✨ Особенности

*   **🔐 Система пользователей:** Регистрация, авторизация и сохранение прогресса.
*   **🎨 Кастомизация:** Выбор цветовой темы интерфей из 7 доступных вариантов. Настройки сохраняются между запусками.
*   **💾 Автовход:** Приложение запоминает последнего пользователя.
*   **📊 Инструменты:**
    *   Стандартный калькулятор.
    *   Конвертер десятичных чисел в двоичные.
    *   Конвертер двоичных чисел в десятичные.
*   **🖥️ Графический интерфейс:** Полностью отрисован на Pygame без использования стандартных оконных элементов ОС.

## 🚀 Установка и запуск

### Предварительные требования

Убедитесь, что у вас установлен **Python 3.8** или выше.

### Шаги установки

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ВАШ_НИК/Universal-Pygame-Calculator.git
    cd Universal-Pygame-Calculator
    ```

2.  Установите необходимые зависимости:
    ```bash
    pip install pygame
    ```

3.  Запустите приложение:
    ```bash
    python main.py
    ```
    *(Замените `main.py` на имя вашего основного файла, если оно отличается)*

## 📂 Структура файлов

При первом запуске приложение автоматически создаст следующие файлы для хранения данных:

*   `users.json` — база данных пользователей (логины и хеши паролей*).
*   `settings.json` — настройки приложения (выбранная тема, последний пользователь).
*   `settings_icon.png` — (опционально) иконка настроек. Если файл отсутствует, иконка не отображается.

> **Примечание:** В текущей версии пароли хранятся в открытом виде в `users.json`. Для продакшена рекомендуется добавить хеширование.

## 🎮 Управление

### Мышь
*   **Левая кнопка мыши (ЛКМ):** Выбор кнопок, переключение между полями ввода, навигация по меню.
*   **Наведение:** Кнопки подсвечиваются при наведении курсора.

### Клавиатура
*   **Ввод текста:** Поддерживается ввод логина и пароля с клавиатуры.
*   **Backspace:** Удаление последнего символа.
*   **Enter:** Подтверждение входа / регистрации / вычисления в калькуляторе.
*   **Tab:** Переключение между полями ввода логина и пароля.
*   **Esc:** Выход из текущего инструмента (калькулятора/конвертера) обратно в профиль.
