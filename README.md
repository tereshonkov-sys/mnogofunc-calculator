# 🧮 Многофункциональный Калькулятор (Pygame)

Десктопное приложение-калькулятор с системой пользователей, конвертером систем счисления и кастомизацией интерфейса. Написано на **Python** с использованием библиотеки **Pygame**.

## ✨ Возможности

*   🔐 **Личный кабинет:** Регистрация, вход и сохранение истории.
*   🎨 **Темы:** 7 цветовых схем на выбор (сохраняются автоматически).
*   🔄 **Конвертер:** Перевод чисел из десятичной системы в двоичную и обратно.
*   ➕ **Калькулятор:** Стандартные арифметические операции.
*   💾 **Автовход:** Приложение помнит последнего пользователя.

## 🚀 Установка и Запуск

### Вариант 1: Для пользователей (через терминал)

1.  Убедитесь, что у вас установлен [Python 3.8+](https://www.python.org/downloads/).
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/tereshonkov-sys/mnogofunc-calculator.git
    cd mnogofunc-calculator
    ```
3.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
4.  Запустите приложение:
    ```bash
    python main.py
    ```

### Вариант 2: Через PyCharm

1.  Откройте проект в PyCharm.
2.  Перейдите в `File` -> `Settings` -> `Project Interpreter`.
3.  Нажмите `+` и установите пакет `pygame`.
4.  Нажмите кнопку **Run** (зеленый треугольник).

## 📂 Структура проекта

*   `main.py` — Основной файл с логикой игры и интерфейсом.
*   `users.json` — Создается автоматически. Хранит логины и пароли.
*   `settings.json` — Создается автоматически. Хранит выбранную тему и последнего пользователя.
*   `settings_icon.png` — (Опционально) Иконка шестеренки для меню настроек.

## ⚠️ Важно

*   Если файл `settings_icon.png` отсутствует, иконка настроек отображаться не будет (это не вызовет ошибку).
*   Пароли хранятся в файле `users.json` в открытом виде. Не используйте важные пароли.
