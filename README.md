# 🧮 Универсальный Калькулятор на Pygame

Это десктопное приложение-калькулятор, разработанное на Python с использованием библиотеки **Pygame**. Проект включает в себя не только базовые вычислительные функции, но и полноценную систему авторизации пользователей, сохранение настроек темы и историю действий.

## ✨ Особенности

*   **Система аккаунтов:** Регистрация и вход в систему. Данные пользователей сохраняются локально.
*   **Персонализация:** Возможность выбора цветовой темы интерфейса (7 цветов на выбор). Настройки сохраняются между запусками.
*   **Авто-вход:** Приложение запоминает последнего пользователя и предлагает быстрый вход.
*   **Калькулятор:** Полноценный графический калькулятор с поддержкой основных арифметических операций (+, -, *, /), скобок и десятичных дробей.
*   **Безопасность данных:** Все данные (пользователи и настройки) хранятся в локальных JSON-файлах.
*   **Адаптивный интерфейс:** Эффекты наведения на кнопки, визуальная обратная связь при вводе текста.

## 🚀 Установка и запуск

### Требования
*   Python 3.6 или выше
*   Библиотека `pygame`

### Инструкция

1.  **Клонируйте репозиторий** или скачайте исходный код:
    ```bash
    git clone <ссылка_на_ваш_репозиторий>
    cd <папка_проекта>
    ```

2.  **Установите зависимости**:
    ```bash
    pip install pygame
    ```

3.  **Запустите приложение**:
    ```bash
    python main.py
    ```
    *(Примечание: Если ваш главный файл называется иначе, например `calculator.py`, замените `main.py` на его имя).*

## 📁 Структура проекта

*   `main.py` — Основной исполняемый файл с логикой игры и интерфейсом.
*   `users.json` — Файл для хранения логинов и паролей зарегистрированных пользователей (создается автоматически).
*   `settings.json` — Файл для сохранения настроек темы и последнего активного пользователя (создается автоматически).
*   `settings_icon.png` — Иконка настроек (опционально, если файл отсутствует, иконка не отображается).

## 🎮 Как пользоваться

1.  **Регистрация:** При первом запуске нажмите кнопку **"РЕГИСТРАЦИЯ"**. Придумайте логин (мин. 3 символа) и пароль (мин. 4 символа).
2.  **Вход:** Используйте свои учетные данные для входа.
3.  **Настройки:** Нажмите на иконку шестеренки в левом верхнем углу, чтобы изменить цвет темы приложения.
4.  **Калькулятор:** В меню профиля выберите кнопку **"Калькулятор"** для перехода к режиму вычислений.
5.  **Выход:** Кнопка **"ВЫЙТИ"** в профиле позволяет сменить пользователя.
